# About This Portfolio

This portfolio was rebuilt to function as a **curated engineering showcase** rather than a gallery of every repository.

The goal is to highlight the **most technically significant projects** while still allowing access to additional work through **GitHub integration**.

The site focuses on:

- Clear **project case studies**
- Demonstrating **full-stack architecture**
- Showing **real-world problem solving**
- Providing recruiters a **quick overview of skills and experience**

# Portfolio Structure

```
app/
components/
sections/
data/
public/
```

### Key Directories

**components**  
Reusable UI components used throughout the site.

**sections**  
Major homepage sections such as Hero, Featured Projects, and Tech Stack.

**data**  
Curated project metadata used to control how projects appear.

**app/projects/[slug]**  
Dynamic project case study pages.

# Key Features

### Curated Project System

Instead of automatically displaying every repository, the portfolio highlights a curated set of projects that demonstrate stronger engineering work.

# Local Development

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open in browser:

http://localhost:3000

# Deployment

The portfolio is deployed using Vercel.

Deployment is automatic when pushing to the main branch.
